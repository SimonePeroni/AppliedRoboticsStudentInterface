BUILDING

cd workspace/project
$ git pull https://github.com/SimonePeroni/AppliedRoboticsStudentInterface
$ mkdir build
$ cd build
$ cmake ..
$ make
$ source ../environment.sh

Usage

$ cd workspace/simulator
$ source ../environment.sh
$ AR_simulator n:=2 # select number of robots, for our case it is two
$ AR_pipeline n:=2 # open it in a new terminal and source environment before

Alternatively to the pipeline method:

$ AR_rviz n:=2 # open it in a new terminal
$ AR_plan n:=2 # open it in a new terminal
$ AR_run n:=2
