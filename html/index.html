Building instructions with CMake

cd workspace/project
$ git pull https://github.com/SimonePeroni/AppliedRoboticsStudentInterface
$ mkdir build
$ cd build
$ cmake ..
$ make
$ source ../environment.sh

Usage

In the main terminal, run the simulator with two robots

$ cd workspace/simulator
$ source ../environment.sh
$ AR_simulator_gui n:=2 # run simulator with two robots

In another terminal, start the pipeline:

$ cd workspace/simulator
$ source ../environment.sh
$ AR_pipeline n:=2

In yet another terminal, run the planning task:

$ cd workspace/simulator
$ source ../environment.sh
$ AR_plan
$ AR_run
